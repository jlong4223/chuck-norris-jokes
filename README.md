## Chuck Norries Joke Generator
An app connected to an API that will generate random Chuck Norries jokes by category with the click of a button. 


### Technologies used: 
- Chuck Norris API
- Node.js
- Express
- Morgan
- EJS
- Javascript
- CSS
- HTML
